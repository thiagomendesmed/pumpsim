# Referencia de Doses — Drogas de Infusao Continua em UTI

## Fontes Principais

- **Surviving Sepsis Campaign (SSC) 2021**: Evans L, et al. *Surviving Sepsis Campaign: International Guidelines for Management of Sepsis and Septic Shock 2021.* Intensive Care Med. 2021;47(11):1181-1247.
- **PADIS 2018**: Devlin JW, et al. *Clinical Practice Guidelines for the Prevention and Management of Pain, Agitation/Sedation, Delirium, Immobility, and Sleep Disruption in Adult Patients in the ICU.* Crit Care Med. 2018;46(9):e825-e873.
- **SOAP II Trial**: De Backer D, et al. *Comparison of dopamine and norepinephrine in the treatment of shock.* N Engl J Med. 2010;362(9):779-789.
- **ACURASYS Trial**: Papazian L, et al. *Neuromuscular blockers in early acute respiratory distress syndrome.* N Engl J Med. 2010;363(12):1107-1116.
- **UpToDate 2024**: Referencia clinica para doses e faixas terapeuticas.
- **Micromedex / Lexicomp**: Bases de dados farmacologicas.

## Formulas de Calculo

### Drogas com dose por peso (mcg/kg/min)
```
Dose (mcg/kg/min) = (Vazao mL/h x Concentracao mcg/mL) / (60 x Peso kg)
Vazao (mL/h) = (Dose alvo x 60 x Peso kg) / Concentracao mcg/mL
```

### Drogas mcg/kg/h (Fentanila, Dexmedetomidina)
```
Dose (mcg/kg/h) = (Vazao mL/h x Concentracao mcg/mL) / Peso kg
```

### Drogas mg/kg/h (Midazolam, Propofol, Cetamina)
```
Dose (mg/kg/h) = (Vazao mL/h x Concentracao mcg/mL) / (1000 x Peso kg)
```

### Drogas com dose fixa — mcg/min (Nitroglicerina)
```
Dose (mcg/min) = (Vazao mL/h x Concentracao mcg/mL) / 60
```

### Drogas com dose fixa — mg/h (Morfina)
```
Dose (mg/h) = (Vazao mL/h x Concentracao mcg/mL) / 1000
```

### Drogas com dose fixa — U/min (Vasopressina)
```
Dose (U/min) = (Vazao mL/h x Concentracao U/mL) / 60
```

## Notas Importantes

1. **Todas as doses sao para adultos** em contexto de terapia intensiva.
2. **Titular por efeito clinico** — as faixas sao orientativas, nao absolutas.
3. **Monitoramento hemodinamico** e obrigatorio para vasopressores e vasodilatadores.
4. **BNM requer sedacao e analgesia** concomitantes — SEMPRE verificar antes de iniciar.
5. **Compatibilidade** — verificar compatibilidade de diluentes (ex: Dexmedetomidina NAO dilui em SG5%).
6. **Protecao da luz** — Nitroprussiato de Sodio e fotossensivel.
7. Este documento e para **fins educacionais**. Sempre seguir protocolos institucionais.
